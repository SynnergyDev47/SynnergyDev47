# Guía de marca

## Identidad visual

- Color principal: `#0a2e5a` (Dark Blue Syn).
- Color secundario: `#7b1ba4` (Purple Syn).
- Color neutro: `#f2f2f2` (White Neutral Syn).
- Tipografía principal para títulos: **Exo 2**.
- Tipografía secundaria para texto corrido: **Montserrat**.

## Uso de logo en la web

Coloca los archivos del logo en `assets/img/branding/` con estos nombres para mantener consistencia:

- `assets/img/branding/synnergy-lab-logo-color.svg`
- `assets/img/branding/synnergy-lab-logo-white.svg`

Si prefieres PNG, usa la misma convención de nombre (`.png`) y mantén una versión a color y otra en blanco. La estructura actual de la web ya deja esa carpeta reservada para branding y assets globales.

## Criterios de implementación

- Responsive first para móvil, tablet, laptop y escritorio.
- Código en inglés para variables, funciones, módulos y nombres técnicos.
- Texto visible y contenido localizado por idioma en carpetas separadas.
- Arquitectura simétrica por idiomas: español en `/es/`, inglés en `/en/` y raíz reservada para resolver el idioma por defecto.
- La marca debe escribirse siempre como **Synnergy Lab**, sin versión corta en la interfaz.
- El mensaje debe presentar una sola entidad: una agencia de marketing estratégico para mipymes.
