# Project image placeholders

This folder contains temporary SVG placeholders for the published Synnergy Lab projects.

- `luminare-photography-placeholder.svg` corresponds to **Luminare Photography**.
- `mr-fox-placeholder.svg` corresponds to **Mr Fox**.

Replace these files with the definitive project photographs when they are available. If the final file name or extension changes, update the matching project path in `data/projects.json` under the `image.src` field.
